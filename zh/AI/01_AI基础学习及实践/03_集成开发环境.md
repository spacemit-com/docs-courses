# 集成开发环境（IDE）

​集成开发环境（Integrated Development Environment，简称 IDE）是一种软件应用程序，旨在为开发者提供全面的开发工具集，以便于软件开发过程中的编写、调试和维护。我们的系统中默认安装了 VSCodium 和 Zed 两款 IDE，本文主要介绍如何使用 VSCodium 来运行 python 和 C++ 代码，但也会在最后附上 Zed 的使用简介。
